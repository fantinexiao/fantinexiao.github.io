# Personal Website

This is a small static personal website built with plain HTML, CSS, and JavaScript.

## Edit Your Details

- Open `index.html`.
- Replace `Your Name`, `YN`, `your.email@example.com`, and the placeholder project text.
- Update the GitHub and LinkedIn links in the contact section.
- Open `index.html` in your browser to preview the site.

## Publish With GitHub Pages

1. Open this folder in GitHub Desktop.
2. Create a repository and commit the files.
3. Publish the repository to GitHub.
4. In GitHub, go to `Settings` -> `Pages`.
5. Choose the `main` branch and the root folder, then save.

GitHub will give you a public website link after Pages finishes deploying.
